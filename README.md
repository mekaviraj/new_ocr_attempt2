### ğŸ¯ Objective  
Develop a layout-aware Optical Character Recognition (OCR) pipeline to extract structured data from handwritten medical prescriptions. The system is tailored to handle medical-specific layouts and convert image-based prescriptions into machine-readable formats.

---

### ğŸ”‘ Key Features of the Layout Aware OCR System  

1. **Prescription Image Input**: Supports image-based prescriptions from scans or mobile photos.  
2. **Image Preprocessing**: Converts to grayscale, blurs, binarizes, and aligns the image for optimal OCR accuracy.  
3. **Region-Based Layout Parsing**: Segments areas like patient details, date, and medicine tables using ROI slicing.  
4. **Custom OCR Function**: Extracts text using Tesseract with layout-specific configurations.  
5. **Medicine Table Parsing**: Uses contour detection to isolate rows, columns (medicine, dosage, frequency).  
6. **Output Structuring**: Converts OCR output into a CSV file with organized prescription details.  
7. **Lightweight Solution**: Can run offline, uses minimal external libraries, ideal for basic deployment.  
8. **Extendable**: Provides a modular pipeline to improve with advanced deep learning models in the future.
